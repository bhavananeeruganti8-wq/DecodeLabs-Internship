# 🤖 DecodeBot - Rule-Based AI Chatbot

## Description
DecodeBot is a simple Rule-Based AI Chatbot developed using Python.

The chatbot responds to predefined user inputs using if-elif-else decision-making logic. It can handle greetings, basic AI-related questions, help commands, and exit commands while running continuously in a loop.

This project demonstrates fundamental Artificial Intelligence concepts such as:

- Input Sanitization
- Decision Making
- Rule-Based Responses
- Continuous Interaction Loop
- Deterministic AI Behavior

## Features

✅ Greeting responses (hi, hello, hey)

✅ Answer simple questions

✅ Help menu with available commands

✅ Handles thank-you messages

✅ greetings

✅ Exit command (bye)

✅ Emoji-enhanced user experience

## How to Run

### 1. Open Terminal

Navigate to the project folder:

```bash
cd ~/Desktop/AI\ INTERNSHIP
```

### 2. Run the chatbot

```bash
python3 decodebot.py
```

### 3. Start chatting

Example:

```text
😊You: hi
🤖Bot: Hello! Nice to meet you.

```
